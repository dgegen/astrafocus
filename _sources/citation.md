# Citation

If you use AstraFocus in your research, please cite it. A DOI is minted via [Zenodo](https://zenodo.org) for each release.

To get an up-to-date citation in your preferred format (BibTeX, APA, etc.), use the **Cite this repository** button on the [GitHub repository](https://github.com/dgegen/astrafocus) — it reads directly from `CITATION.cff` and always reflects the latest release.
